# Computação, Sociedade e Sustentabilidade

💻 Projeto de extensão universitária voltado ao desenvolvimento de soluções digitais e estratégias sociais para apoiar instituições locais, incluindo a prototipagem de um site para o LARBEM e a organização de iniciativas de captação de recursos para o grupo Partilhar.

-------------------------------

🎈 Este repositório reúne as atividades desenvolvidas em um projeto de extensão universitária com foco em impacto social, envolvendo duas instituições: LARBEM e Grupo Partilhar. Para o LARBEM, o projeto consiste na prototipagem e planejamento de um site institucional, com foco em usabilidade, acessibilidade e organização clara das informações, seguindo a identidade visual da instituição. O objetivo é facilitar o acesso do público às informações e melhorar a experiência de navegação.

Já para o Grupo Partilhar, as ações estão voltadas para a criação de estratégias de captação de recursos, incluindo a organização de eventos beneficentes, bazares, oficinas e minicursos. O projeto também envolve planejamento de divulgação, engajamento do público e uso de tecnologia para facilitar doações. Além disso, o repositório inclui materiais de planejamento, fluxogramas, protótipos e estratégias que apoiam o desenvolvimento das soluções propostas.
