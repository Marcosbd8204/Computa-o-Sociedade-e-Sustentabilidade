*Nesta partes iremos fazer o relatorio de nosso projeto para beneficiar a casa de acolhimento "Lar Bem"

**Base:
 - Título e Descrição: Um nome claro seguido de um resumo do que o projeto faz.
 - Badges: Ícones pequenos que indicam status de build, licença ou tecnologias (use Shields.io).
 - Funcionalidades (Features): Liste o que o projeto faz com checkboxes.
 - Pré-requisitos e Instalação: Instruções passo a passo sobre como baixar, configurar e rodar o projeto.
 - Como Usar: Exemplos práticos, com códigos ou GIFs, mostrando o projeto em ação.
 - Tecnologias: Ícones ou lista das linguagens e frameworks usados.
 - Contribuição: Guia de como outros desenvolvedores podem ajudar.
 - Licença: Defina como o projeto pode ser usado
