# Nesta pasta iremos fazer o relatorio de nosso projeto para beneficiar a casa solidaria "Casa Partilhar"  
_____________________________________________________________________________________________________
## Base:
1. Título e Descrição: Um nome claro seguido de um resumo do que o projeto faz.
2. Badges: Ícones pequenos que indicam status de build, licença ou tecnologias (use Shields.io).
3. Funcionalidades (Features): Liste o que o projeto faz com checkboxes.
4. Pré-requisitos e Instalação: Instruções passo a passo sobre como baixar, configurar e rodar o projeto.
6. Tecnologias: Ícones ou lista das linguagens e frameworks usados.
7. Contribuição: Guia de como outros desenvolvedores podem ajudar.
8. Licença: Defina como o projeto pode ser usado.
